# RB Team - Sajt za bezbednost i zdravlje na radu

Sajt za kompaniju RB Team koja se bavi bezbednošću i zdravljem na radu.

## Struktura projekta

```
.
├── index.html      # Glavna HTML stranica
├── styles.css      # CSS stilovi
├── script.js       # JavaScript funkcionalnost
├── images/         # Folder za slike (logo)
└── README.md       # Dokumentacija
```

## Instalacija i pokretanje

1. Postavite logo firme u folder `images/` sa imenom `logo.png`
2. Otvorite `index.html` u web pregledaču
3. Ili koristite lokalni server (npr. Live Server u VS Code)

## Funkcionalnosti

- ✅ Responsive dizajn (prilagođen za mobilne uređaje)
- ✅ Prekidač jezika (Srpski/Engleski)
- ✅ Smooth scroll navigacija
- ✅ Mobilni meni (hamburger meni)
- ✅ Kontakt forma
- ✅ Sekcije: Početna, O nama, Karijera, Kontakt, Vesti

## Postavka loga

Postavite logo firme u folder `images/` sa tačnim imenom `logo.png`. Logo će se automatski prikazati u navigacionom meniju.

## Prilagođavanje sadržaja

Svi tekstovi su definisani u `script.js` fajlu u objektu `translations`. Možete lako izmeniti:
- Kontakt informacije (adresa, telefon, email)
- Tekstove sekcija
- Dodati nove sekcije

## Napomene

- Kontakt forma trenutno samo prikazuje alert poruku. Za funkcionalnu formu, potrebno je dodati backend server.
- Logo treba biti u formatu PNG sa transparentnom pozadinom za najbolji izgled.
